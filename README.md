# 📚 마지막 페이지 (The Last Page)

> **작품은 독자의 감상으로 완성됩니다.**  
> 당신의 감상이 책의 마지막 페이지를 장식합니다.

---

## ✨ 소개

**마지막 페이지**는 책을 읽고 난 뒤,  
당신만의 감상을 깊이 있게 정리할 수 있도록 도와주는 독서 기록 앱입니다.  

단순한 기록이 아닌, 책의 핵심 키워드와 생각을 자극하는 질문을 통해  
읽은 내용을 다시 바라보게 합니다.  
책을 읽고 어떤 점이 마음에 남았는지,  
읽기 전과 후의 생각이 어떻게 달라졌는지,  
그 모든 감상이 **마지막 페이지**에 담깁니다.  

---

## 🎯 기획 의도

- 책의 **의도를 온전히 이해하고**, 더 깊이 생각해볼 수 있기를 바랍니다.  
- 단순히 ‘읽었다’고 적는 것을 넘어서, **내 생각의 변화**를 기록하길 바랍니다.  
- 책을 통해 **세상을, 그리고 나 자신을** 들여다볼 수 있기를 바랍니다.

---

## ✨ 감상 포인트

### 1. 📚 감상의 방향을 잡아주는 키워드  
**핵심 키워드와 질문**이 감상의 실마리를 던져줘요.  
“왜 이런 방식으로 썼을까?” “이건 어떤 의미일까?”  
**생각이 하나 둘 연결되며, 깊이가 생깁니다.**  

---

### 2. ✍️ 읽기 전의 나 vs 읽은 후의 나  
책을 읽기 전의 기대도 기록하고, 읽고 난 후 변화된 생각을 비교해 보세요.  
“내가 이 책을 왜 골랐더라?” “이 책을 읽고 나는 어떤 변화가 있을까?”  
**생각의 변화를 기록하는 것만으로도, 의미가 달라집니다.**  

---

### 3. 📷 꽂힌 문장은 바로 저장  
책을 읽다가 마음을 건드린 문장,  
**카메라로 딱 스캔하면 메모가 완성돼요.**  
손으로 적느라 흐름을 놓치지 말고,  
**그때의 감상을 빠르게 기록해보세요.**

## 🧩 기술 스택

- **Architecture**: MVVM / Clean Architecture / DI / Coordinator
- **Reactive**: Combine (Input / Output 구조)
- **DB**: Realm
- **UI**: SnapKit, Kingfisher
- **API**: Aladin API, ChatGPT API (키워드 추천)
- **기타**: VisionKit (OCR 활용), NaturalLanguage

## 🖼️ 주요 화면 구성

### 🏠 홈
- 저장된 도서를 프리뷰
- 최근 저장 도서 강조
- 검색 및 도서 추가 버튼 제공

### 🔍 검색
- 제목 또는 저자, isbn 등으로 도서 검색
- 도서 클릭 시 기록 뷰로 이동
- 상단 ‘+’ 버튼으로 새로운 도서 기록 작성

### 📖 독서 기록
- 도서 정보 표시 및 수정
- 상태별 기록 분류 (읽기 전/중/후)
- ‘스캔’ 버튼으로 구절 텍스트 추출 → 메모 자동 생성
- ‘휴지통’ 버튼으로 도서 삭제

### ✏️ 도서 정보 수정
- 책 표지 수정 (앨범에서 가져오기)
- 제목/저자 필수 입력
- 독서 상태, 카테고리, 감정 태그 입력

### 🗒 도서 기록 수정
- 상태에 따른 기록 입력/수정
- ‘읽는 중’에서는 메모 추가/삭제, 구절 스캔 가능
- ‘읽은 후’에서는 키워드 도움받기 가능

### 📂 도서 보관함
- 저장된 도서 리스트
- 상태, 카테고리, 감정 태그로 필터링
- 실시간 검색 가능

### 📊 통계
- 월/년 단위 통계
- 달력 기반 날짜별 도서 기록 조회

### ⚙️ 설정
- Help & FAQ
- 앱 정보 (About)
- 초기화 (Reset)

책을 읽고, 마지막 페이지를 장식해보세요.
